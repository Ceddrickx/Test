# Loan-Amortization-Calculator

A Java-based Loan Amortization Calculator that accurately computes monthly payments, total interest, and overall loan cost. The application features a user-friendly Java Swing graphical interface, automated calculations, and detailed amortization schedule generation to support efficient personal and educational financial planning.

---

## 📌 Features

- ✅ User-friendly Java Swing GUI  
- ✅ Automatic monthly payment calculation  
- ✅ Total interest and total loan cost computation  
- ✅ Detailed amortization schedule table  
- ✅ Input validation for numeric and positive values  
- ✅ Supports zero-interest loan scenarios  
- ✅ Clear/reset functionality for quick recalculation  
- ✅ Offline desktop application  

---

## 🛠️ Built With

- **Java**
- **Java Swing**
- **Object-Oriented Programming (OOP)**
- **Standard Amortization Formula**

---

## 🎯 Project Objectives

- Accurately compute monthly loan payments  
- Calculate total interest paid and total loan cost  
- Demonstrate fundamental programming concepts  
- Provide a simple and intuitive financial planning tool  
- Reduce errors from manual loan computation  

---

## 📊 How It Works

1. Enter the following loan details:
   - Loan Amount (Principal)  
   - Interest Rate  
   - Loan Term  

2. Click the calculate button.

3. The system will display:
   - Monthly payment  
   - Total interest paid  
   - Total loan cost  
   - Full amortization schedule  

---

## ⚠️ Scope and Limitations

- ❌ No database (data is temporary)  
- ❌ No export to PDF/Excel yet  
- ❌ Single currency support (PHP only)  
- ❌ Fixed-rate loans only  
- ❌ No real-time bank rate integration
